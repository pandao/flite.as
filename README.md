# FLite.as

A simple ActionScript 3.0 class libraries for Flash/Flex (website) application.

####Classes

- Animation
- Debuger
- FullscreenBackground
- ImageLoader
- Scrollbar
- Utils
- xContextMenu
- XMLLoader
